# Análise de Corridas de Táxi em Chicago

## Descrição

Projeto de análise de dados utilizando SQL para extração das informações e Python para análise exploratória, visualização de dados e teste de hipóteses.

## Objetivos

- Analisar empresas de táxi de Chicago
- Identificar bairros mais populares para corridas
- Investigar padrões de viagens
- Testar hipóteses sobre a duração das corridas

## Tecnologias Utilizadas

- SQL
- PostgreSQL
- Python
- Pandas
- Matplotlib
- SciPy
- Jupyter Notebook
